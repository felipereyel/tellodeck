# tellodeck
Control your DJI Tello with your Steamdeck

## Setup

Sync uv: `uv sync`
Build: `uv run pyinstaller main.py`

## Controls

START: Takeoff
SELECT: Land
Horizontal movement: Use right axis
Vertical + rotation: Use left axis
